# Control Center - 통합 관리 모니터링 시스템

## 📁 프로젝트 구조

```
Control_center/
├── index.html              # 메인 진입점
├── assets/                 # 정적 리소스 (이미지, 아이콘, 폰트)
├── styles/                 # CSS 스타일
│   ├── base/              # 기본 스타일 (reset, variables, typography)
│   ├── components/        # 컴포넌트별 스타일
│   ├── layouts/           # 레이아웃 (desktop, mobile)
│   ├── pages/             # 페이지별 스타일
│   └── main.css           # CSS 진입점
├── js/                    # JavaScript
│   ├── core/              # 핵심 기능 (app, config, utils)
│   ├── services/          # API/데이터 서비스
│   ├── components/        # UI 컴포넌트
│   ├── pages/             # 페이지 모듈
│   ├── stores/            # 상태 관리
│   └── main.js            # JS 진입점
└── data/                  # 목업/샘플 데이터
```

## 🚀 시작하기

로컬 서버를 실행하여 프로젝트를 확인할 수 있습니다:
```bash
npx serve .
```
